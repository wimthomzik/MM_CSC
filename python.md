# how to run the python scripts

## setup

```sh
# Create a new virtualenv
pip -m venv venv

# enter the venv
source ./venv/bin/activate

# install dependencies
pip install -r ./requirements.txt
```

## run a script

```sh
# enter the venv, if you haven't already
source ./venv/bin/activate

# run it
python <script path>
```
